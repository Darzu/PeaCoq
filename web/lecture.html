Help and shortcuts

          

          

            

              In all the following, Mod will stand for any combination of Ctrl and other keys, so pick whichever combination is not captured by your desktop environment (Ctrl, Ctrl+Alt, Ctrl+Shift, ...).
            

            

            

              

                
Editor mode

                
    
                  
  • 
                    Mod
                    +
                    ↓
                    : Process next command
                  
    • 
                  
  • 
                    Mod
                    +
                    ↑
                    : Unprocess last command
                  
    • 
                  
  • 
                    Mod
                    +
                    Enter
                    : Process or unprocess to caret
                  
    • 
                

              

              

                
Proof Tree mode

                
    
                  
  • 
                    →
                    : Enter the focused subtree
                  
    • 
                  
  • 
                    ←
                    : Backtrack to the parent goal
                  
    • 
                  
  • 
                    ↓/↑
                    : Browse tactic groups, or browse remaining subgoals
                  
    • 
                  
  • 
                    [/]
                    : Browse tactics within a tactic group
